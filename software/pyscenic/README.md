# Running pySCENIC

This directory contains the code required to set up and run pySCENIC.

## Prerequisites

- `conda`
- `jupyter`

## Setting up the pySCENIC `conda` environment

If a pySCENIC environment does not already exist, run the code below to create the environment and activate it.

```
$ conda env create -f pyscenic.yml
$ conda activate pyscenic
```

## pySCENIC template

The Jupyter Notebook `PBMC10k_SCENIC-protocol-CLI.ipynb` can be used as a template for running pySCENIC (as suggested by Aerts lab).

The `pySCENIC_G34_MB.ipynb` was generated by Dapo (rotation student).

## Analysis steps

### Normal human RL (Aldinger data)

1. Save Seurat counts and metadata as CSV for use in downstream SCENIC analysis
    - [ ] `seurat_to_scenic.R`
1. pySCENIC on Aldinger RL-derived cells
    - [ ] `aldinger_RL.ipynb` (Note: not tracked with git cuz file is too big to push to GitHub. Edit then copy to results folder if needed.)
1. Make Venn diagram of regulators (i.e. TFs) for different cell types (e.g. RL-VZ vs RL-SVZ)
    - [ ] `shared_cell_type_tfs.R`
1. Make Venn diagram of regulators (i.e. TFs) that were found to be mutated in MB
    - [ ] `shared_pyscenic_tfs_mb_mut.R`
1. Extract the genes that are in each regulon and save to CSV files
    - [ ] `get_genes_in_regulon.py`

### Human tumour (Vladoiu data)

1. Save Seurat counts and metadata as CSV for use in downstream SCENIC analysis
    - [ ] `seurat_to_scenic.R`
1. pySCENIC on Vladoiu medulloblastoma cells
    - [ ] `vladoiu_mb.ipynb` (Note: not tracked with git cuz file is too big to push to GitHub. Edit then copy to results folder if needed.)
