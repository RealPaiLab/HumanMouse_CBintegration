# List of scripts

These scripts contain useful functions that may be called from other scripts.

## `check_feature_expression.R`

Convenience functions that generates various gene expression plots and saves them.

## `convert_genes.R`

Functions to identify orthologous mouse/human genes using BioMart.

## `initial_visual_check.R`

Script to generate basic visualizations of a Seurat object.

## `process_raw_seurat.R`

Convenience functions to process a basic Seurat object containing only raw counts and to integrate a list of Seurat objects.
