# List of scripts

These scripts were used to analyze the Aldinger single-cell data.

## `Aldinger-trajectory.R`

Old code from rotation for pseudotime analysis.

## `Aldinger-visualization.R`

Old code from rotation for visualizing the dimensionality reductions and clustering.

## `isolate_UBC.R`

Code to isolate UBCs from the RL for downstream differential expression analysis.

## `load_seurat.R`

Loads the original Aldinger Seurat object (entire cerebellum) and Liam's Seurat object (RL only)

## `run_pct_cells_gene_pos.R`

Checks percentage of cells that express a specific gene (namely, *EOMES*).
